# P1 H-Section FE Representation Sensitivity - Reproducibility Package v8

This package contains non-sensitive reproducibility assets for a bounded manuscript on shell finite-element representation sensitivity in H-section steel members under in-plane shear.

## What is included

- Public derived CSV audit tables.
- A solver-status summary figure generated from derived audit data.
- Traceability metadata for the v8 supplement.
- Local audit script used to generate the internal v8 supplement.
- Runbook, dataset/source registry and citation metadata.

## What is excluded

- Raw Abaqus `.odb`, `.cae`, `.sim`, `.msg`, `.sta`, `.dat`, `.log`, `.prt` and other solver files.
- Active submission manuscript, cover letter, reviewer drafts and internal logs.
- Third-party raw data or files with unclear redistribution rights.

## Evidence boundary

The package supports auditability of available derived results and solver-file traces. It does not claim experimental validation, ultimate load, design resistance or a deconfounded shell-representation effect.
