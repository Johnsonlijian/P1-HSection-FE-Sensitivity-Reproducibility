Raw Abaqus binary outputs are not redistributed because they contain large project-internal solver files. The public repository includes derived non-sensitive tables sufficient to reproduce the reported ratios, confidence intervals, claim-boundary table and solver-status summaries.

## Reproducibility Statement

The public reproducibility package contains derived CSV tables for the 11-pair elastic eigenvalue-ratio audit, the secondary nonlinear RF-U response-index audit, model-parameter registry summaries, the claim-boundary table and public PNG versions of the main figures. The package is an audit supplement rather than a full Abaqus rerun archive because raw `.odb`, `.cae`, `.sta`, `.msg` and related solver files are not redistributed.
